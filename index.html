hyde.js - device fingerprinting assessment platform

hyde.js is a platform build on meteor.js (@meteor) which provides an easy-to-use framework for testing device fingerprinting algorithms, tools and libs. You can easily add in new fingerprinting techniques and examine their performance. To start the meteor app just type:

$ cd your_repo_root/repo_name
$ meteor

Development

hyde.js is currently under heavy development and includes following features:

  • fingerprint.js2 (@Valve)
  • user authentication
  • (optional) saving to database
  • admin dashboard (@yogiben)
  • bootstap3
  • ...

more is coming the next few weeks.
